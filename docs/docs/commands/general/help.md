# Help Command

Display an overview of available commands and features.

## Description

The /help command displays more information concerning a specific command or feature. It also provides links to this
documentation and support pages.

Please note that the contents of the /help command are automatically generated and provide a less detailed overview
of specific commands.

## Usage

```asm
/help [<command>]
```

Parameters:

- `command` | Shows information about a specific command.

## Examples

```asm
# Get information and usage for the /hug command
/help hug

# Get information about all available commands
/help
```
