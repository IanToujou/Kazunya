# Pat Command

Interact with another user by giving them headpats.

## Description

The /pat command will give another user headpats. Kazunya will return a matching GIF and ping the mentioned user.
GIFs are selected depending on the selected mode and personal settings.

Please note that the command can fail under certain circumstances, for example, if they disabled pats.
More information, including how to manage your preferences, can be found in the [Roleplay]() section.

## Usage

```asm
/pat <user>
```

Parameters:

- `user` | The user you want to cuddle.

Additional settings, such as GIF appearance, whitelisting, etc. can be configured on the [Personal Dashboard]().

## Examples

```asm
# Give Kazunya some headpats!
/pat @Kazunya
```
